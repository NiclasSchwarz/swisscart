Content Sub-Template

Here starts the product info sub-template which replaces the $content variable of the main template. The sub-template can be found in your templates's sub-folder named content.

Note: The product info sub-template does currently neither support additional images nor the so called option type feature that allows you to select option rendering other then the default drop down selector.
